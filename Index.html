Introduction

Transitions en CSS3 offre une méthode simple d'animer des propriétés CSS d'un élément HTML, permettant de créer un rollover, fade etc… sans avoir recours à Javascript.
Une transition est moins complexe à implémenter qu'une animation mais ne propose pas le même niveau de contrôle. On retiendra que Transition sert à créer une transition fluide pour passer d'un état A à un état B. Animations en CSS3 permet d'ajouter des animations à du contenu HTML basées sur le keyframe. Le keyframe est le principal composant des animations en CSS Considérez les keyframes comme des étapes sur une timeline. A l'intérieur d'une keyframe, vous allez pouvoir définir ces étapes, et lier ces keyframes à un selecteur. Animations permet un contrôle pointu sur la façon dont les propriétés CSS d'un élément HTML va réagir sur une page
